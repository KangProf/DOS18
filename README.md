# DOS18
DOS18 adalah tools ddos yang terbuat dari perl, fungsinya untuk membanjiri website sehingga itu tidak bisa di akses/DOWN

# Cara Menggunakan

$ pkg update && pkg upgrade

$ pkg install git

$ pkg install python2

$ pkg install perl

$ git clone https://github.com/KangProf/DOS18

$ cd DOS18

$ ls

$ perl DoS18.pl

# Perhatian!!
